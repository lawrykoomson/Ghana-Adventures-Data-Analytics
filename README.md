# 🌍 Ghana Adventures — Data Analytics Project

## Project Overview
A complete end-to-end data analytics project built in Power BI for 
Ghana Adventures, a travel and tour company based in Tamale, Ghana.
This project was developed as a Data Analyst interview case study.

## Business Problem
Ghana Adventures was experiencing inconsistent bookings, low customer 
retention and underperforming tour packages. The goal was to analyse 
their booking data and deliver actionable insights to improve revenue 
and customer retention.

## Dataset
- Source: Ghana Adventures booking records
- Period: October 2024 — February 2025
- Records: 91 clean bookings after data cleaning
- Columns: Date, Customer ID, Tour Package, Destination,
  Price (GHS), Booking Type, Repeat Customer, Season, Booking Channel

## Key Findings
- Total Revenue: GHS 63,250 across 91 bookings
- Average Booking Value: GHS 695 per booking
- Repeat Customer Rate: 45.05%
- Top Tour: New Year Safari — GHS 3,600
- Top Destination: Mole National Park — GHS 12,000
- Weakest Month: November — GHS 4,250
